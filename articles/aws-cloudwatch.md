# AWS CloudWatchを中学生向けに解説！

AWS CloudWatchは、**インターネット上のシステムを見守る監視カメラ＋健康診断ツール**みたいなものです！

## 身近な例で考えてみよう

学校の保健室を想像してみてください：

- **体温計**で熱を測る
- **体重計**で体重を記録する
- **健康カード**に記録を残す
- 異常があったら**先生に通知**する

CloudWatchも同じように、インターネット上で動いているシステム（ウェブサイトやアプリなど）の「健康状態」をチェックしています。

## CloudWatchが具体的にやっていること

**1. データを記録する（ログ）**
「今日は何人がウェブサイトを見たか」「システムが何回エラーを出したか」といった情報を日記のように記録します。

**2. グラフで見やすくする**
記録したデータを体温計のグラフみたいに視覚化して、「今日は調子いいな」「あれ、おかしいぞ」と分かりやすくします。

**3. アラームを鳴らす**
設定した基準を超えると、スマホのアラームみたいに通知を送ります。例えば「ウェブサイトが止まった！」「アクセスが急増している！」など。

## なぜ必要なの？

例えば、あなたが人気YouTuberのライブ配信を見ているとします。もし配信サーバーに問題が起きても、CloudWatchがすぐに技術者に知らせるので、素早く直せるんです。見守り役がいないと、問題に気づくのが遅れちゃいますよね。

つまり、**24時間365日、システムを見守ってくれる頼れる相棒**がCloudWatchなんです！
