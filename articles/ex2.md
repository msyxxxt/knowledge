# AWS EC2（Amazon Elastic Compute Cloud）とは

AWS EC2 は、クラウド上で仮想サーバーを作成・利用できるサービスです。  
物理サーバーを購入せず、必要なときに必要な性能のサーバーをすぐに使えます。

## 主な特徴

- Linux / Windows サーバーを数分で起動可能
- CPU・メモリ・ストレージを用途に応じて選択できる
- Webサーバー、バッチ処理、検証環境など幅広く利用可能

## メリット

- **従量課金**：使った分だけ支払う
- **スケーラビリティ**：性能変更や台数増減が容易
- **柔軟性**：検証環境から本番環境まで同じ構成で運用可能

## まとめ

AWS EC2 は  
**「物理サーバーを持たずに、必要なときに必要な分だけ使える仮想サーバー」**  
を実現するサービスです。
